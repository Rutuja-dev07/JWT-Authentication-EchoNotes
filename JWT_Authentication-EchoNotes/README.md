# JWT Authentication using Node.js & Express 🔐

This project demonstrates basic JWT Authentication using Node.js and Express.

## Features
- User Signup API
- User Login API
- JWT Token Generation
- Protected Routes using Middleware

## Tech Stack
- Node.js
- Express.js
- JSON Web Token (JWT)

## Folder Structure

routes/ → API routes  
controllers/ → Business logic  
middleware/ → JWT verification
